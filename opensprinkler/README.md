# <img src="icon.png" width="48" /> OpenSprinkler

The official OpenSprinkler Unified Firmware running as a Home Assistant Add-on.

## Features
- **Full UI**: Access the complete OpenSprinkler interface via the sidebar (Ingress).
- **Direct Access**: Optional direct port mapping for mobile apps.
- **Smart Config**: Preserves your password and stations across updates.

## Getting Started
1. **Install** this add-on.
2. Go to the **Configuration** tab and set your `admin_password`.
3. Click **Start**.
4. Click **Open Web UI**.
